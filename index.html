Microsoft прибарахлился Гитхабом за $7.5 млрд
05/06/2020 Новости

Невероятно, но факт — как говорит BBC, Microsoft заключила соглашение о покупке GitHub за 7.5 миллиардов баксов.

В 2015 году знатоки оценивали GitHub в $2 млрд, но с получением прибыли не очень складывалось. Хотя по данным CNBC в прошлом году GitHub планировал таки получить более $200 лямов от платных сервисов для разработки проектов с закрытым кодом. И вот, вместо выхода на IPO владыки Гитхаба решили впарить его за некислую сумму в $7.5 миллиарда. Сборка компа для программиста
03/06/2020 Гайды

Нужен ли мощный компьютер, чтобы кодить? Подойдёт ли ваш домашний компьютер? Вот об этом сейчас поговорим.

Для программирования нужно немного: по сути это написание текста. Компиляция или исполнение кода по современным меркам тоже не особо ресурсоёмкая задача. Поэтому для кодинга подойдёт почти любой компьютер.
Есть лишь три сценария, когда вам потребуется мощный компьютер:
1)Вы делаете сервер, на котором будет обслуживаться много клиентов. Например, вам нужен компьютер, на котором станет крутиться телеграм-бот, а пользоваться им должны десятки тысяч человек постоянно. Тогда нужно много памяти и шустрые многопотоковые процессоры.
2)Вы занимаетесь машинным обучением.
3)Вы используете компьютер не только для кодинга, но и для других задач, например, играете в игры.
